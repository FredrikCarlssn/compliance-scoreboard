# Hackaton - Compliance Scoreboard

Vi är många som jobbar i organisationer som behöver följa samma eller liknande lagstiftning. Hur väl vi uppfyller dessa är generellt olika mellan organisationer och/eller mellan team och område inom en organisation.

Med inspiration från gameing, idéer från GitHub, Swedbank och Omegapoint så vill jag med detta hackaton ta fram ett utkast till hur vi skulle kunna visualisera vår Complience-status i vår kod på ett roligare och mer lustfyllt sätt.

## Upplägg

- Vi börjar med en **introduktion och lite inspiration** - jag visar lite exempel som jag tror vi kan inspireras av!
- **Hackaton** - vi delar in oss i grupper med lite olika fokus: design, utveckling, lagar och mätningar.
- **Metod** - vi jobbar en bit och demar ofta mellan grupperna.
- Koden och designen kommer publiceras med Apache 2 licens och läggs upp GitHub.
